
NSCLC-CNS 风险分层 Demo v1.0
================================

一、这是什么？
这是一个用于老板汇报、专家沟通和项目演示的 Streamlit Demo。
它包含三个模块：

1. A模型：诊断时脑转移风险评估
2. C模型：脑转移后12个月全因死亡风险评估
3. A-C变量联动图：脑转移发生风险 × 脑转移后死亡风险

重要说明：
当前版本是“科研原型展示工具”，不是临床诊断工具，不替代 MRI，不用于治疗决策。

二、如何运行？

推荐在 Anaconda PowerShell Prompt 中运行。

1. 解压本压缩包
2. 进入文件夹，例如：

cd C:\Users\Administrator\Desktop\NSCLC_CNS_Demo_v1

3. 安装依赖：

pip install -r requirements.txt

4. 启动 Demo：

streamlit run app.py

或者直接双击：

RUN_DEMO.bat

三、如何给别人演示？

启动后浏览器会打开本地地址，一般是：

http://localhost:8501

注意：
这个地址默认只在你自己的电脑上可见。
如果要在其他电脑演示，需要把整个文件夹拷贝到那台电脑，并在那台电脑上运行。

四、推荐演示话术

“这个 Demo 不是临床诊断工具，而是我们基于 SEER 公开数据完成的 CNS 风险分层原型。
它分成两个模型：A模型判断诊断时脑转移风险，C模型判断已有脑转移患者的12个月死亡风险。
我们希望用它作为专家沟通工具，进一步设计中国真实世界验证研究。”
